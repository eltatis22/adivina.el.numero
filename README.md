# adivina.el.numero
primer acercamiento a python y en este juego basico la computadora eligira un numero al azar y el usuario tendra que adivinar ese numero

Conceptos practicados
- Condicionales (`if`, `elif`, `else`)
- Bucles (`while`)
- Variables
- Números aleatorios con `random`

▶ Cómo usar
1. Ejecuta el archivo.
2. Ingresa un número.
3. El programa te dirá si es más alto o más bajo.
4. Seguí intentando hasta adivinarlo.
